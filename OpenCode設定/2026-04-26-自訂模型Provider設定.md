# OpenCode 桌面版：新增自訂模型 Provider

## 設定檔位置
`~/.config/opencode/opencode.jsonc`

## 步驟

### 1. 從 disabled_providers 移除目標廠商
```jsonc
// 找到這一行，刪掉 "minimax"
"minimax", "minimax-cn", ...
// 改成
"minimax-cn", ...
```

### 2. 在 provider 區塊新增
```jsonc
"minimax_official": {
  "name": "MiniMax M2.7",
  "npm": "@ai-sdk/openai-compatible",
  "options": {
    "baseURL": "https://api.minimax.io/v1",
    "apiKey": "你的API Key"
  },
  "whitelist": ["MiniMax-M2.7"],
  "models": {
    "MiniMax-M2.7": {
      "name": "① MiniMax M2.7｜說明文字"
    }
  }
}
```

### 3. 重啟 OpenCode 即生效

## 注意：disabled_providers 必須同步移除
只加 provider 不夠，還要把廠商名稱從 disabled_providers 陣列移除，否則不會出現。

## System Prompt 位置
`~/.config/opencode/system-prompt.md`
opencode.jsonc 內有一行指向它：
```jsonc
"instructions": ["~/.config/opencode/system-prompt.md"]
```
